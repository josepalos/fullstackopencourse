## Run the app

### Simple version
Run the app with:

```
$ cd anecdotes/frontend
$ npm start
```

### Redux version
Run the backend server with

```
$ cd courses/frontend
$ npx json-server --port 3001 --watch db.json
```

Start the app using

```
$ cd anecdotes/frontend-redux
$ npm start
```
